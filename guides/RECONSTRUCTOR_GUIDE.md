# Research Content Reconstructor

Combines separated parsed markdown files from research-gatherer into comprehensive, organized documents.

## 🎯 What It Does

The `content_reconstructor.py` script takes the scattered parsed content from research-gatherer and creates:

1. **SUMMARY.md** - Statistical overview of all content
2. **CURATED_ESSENTIALS.md** - High-quality content from official sources only
3. **BY_CATEGORY.md** - Content organized by topic categories
4. **BY_DOMAIN.md** - Content organized by source domain

## 📊 Features

### **Intelligent Filtering**

Automatically filters out low-quality content:
- ❌ Exam dump sites
- ❌ Practice test sites  
- ❌ Job listing pages
- ❌ 404 error pages
- ❌ Very short content (<100 chars)

### **Smart Categorization**

Organizes content into categories:
- 📘 **Official Documentation** - Docs from paloaltonetworks.com, pan.dev
- 📝 **Technical Blogs** - Blog posts and technical articles
- 🔒 **Security News** - Threat intelligence and vulnerability reports
- 📦 **Product Information** - Product specs and features
- 🛠️ **Implementation Guides** - Configuration and deployment guides
- ⚖️ **Comparisons & Reviews** - Product comparisons and reviews

### **Quality Prioritization**

Curated essentials focus on:
- Official documentation (highest priority)
- Technical blogs
- Implementation guides
- Larger, more comprehensive documents

## 🚀 Usage

### **Quick Start - Export Everything**

```powershell
python content_reconstructor.py "research_output\Palo Alto NGFW" --all
```

This creates a `reconstructed/` folder with all 4 export formats.

### **Export Specific Formats**

```powershell
# Summary only
python content_reconstructor.py "research_output\Palo Alto NGFW" --summary "summary.md"

# Curated essentials only (recommended for review)
python content_reconstructor.py "research_output\Palo Alto NGFW" --curated "essentials.md"

# Organized by category
python content_reconstructor.py "research_output\Palo Alto NGFW" --by-category "by_category.md"

# Organized by domain
python content_reconstructor.py "research_output\Palo Alto NGFW" --by-domain "by_domain.md"
```

## 🔧 Customization

### **Modify Category Keywords**

Edit `content_reconstructor.py` line with `self.categories`:

```python
self.categories = {
    'Official Documentation': ['docs.paloaltonetworks', 'pan.dev'],
    'Your Custom Category': ['keyword1', 'keyword2'],
}
```

### **Adjust Quality Filters**

Edit `self.skip_domains` to filter more/less aggressively:

```python
self.skip_domains = [
    'exam', 'dump', 'cert',  # Remove to include exam content
    'jobs', 'careers',        # Remove to include job postings
]
```

### **Change Document Limits**

In `export_curated_essentials()`, adjust the limit:

```python
for doc in docs_sorted[:20]:  # Change 20 to get more/fewer docs
```
